# Tiny URL Shortener Frontend

Frontend application for generating short URLs and viewing analytics.

## Features

- Modern landing page
- URL shortening
- Copy shortened links
- Analytics dashboard
- Click tracking visualization
- Device breakdown charts
- Responsive design

---

## Tech Stack

- React
- TypeScript
- Vite
- React Query
- Axios
- Recharts
- Tailwind CSS
- Radix UI

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Samanvith20/tiny-shortern-url-frontend.git

```

Install dependencies:

```bash
npm install
```

Create environment file:

```env
VITE_API_URL=http://localhost:5000
```

---

## Run Locally

Development:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview Production Build:

```bash
npm run preview
```
